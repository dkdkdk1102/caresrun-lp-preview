# 本番担当向け 申し送り（ケアスルンLP）

このリポジトリは**確認用（preview）として作られた静的サイト**です。コード・画像は全て揃っており、ビルド不要でそのまま配信できますが、**本番稼働の前に下記①〜④の対応が必要**です。

## 構成（このまま動きます）
- `index.html` … LP本体（スマホ縦・画像スタック＋一部HTML）
- `style.css` … 表示/CTA/FAQ/追従CTA/フッター/注記のCSS
- `assets/` … セクション画像13枚（`01_fv`〜`15_closing`）
- ビルドツール・依存パッケージなし。**clone → 静的ホスティングに置くだけ**で表示されます。
- 外部依存（実行時にネット必要）：Google Fonts（Noto Sans JP）／YouTube埋め込み。

## 本番前にやること（重要度順）

### ① CTAリンクを実際の申込先URLに差し替え【最重要】
現在、全CTAの遷移先は `href="#offer"`（ページ内アンカー＝スクロールのみ）で、**購入導線が未接続**です。
- 対象: `index.html` 内の `data-cta` を持つ `<a>`（`fv` / `solution` / `offer` / `closing` / `sticky`）と、追従バーのボタン。
- 対応: `href="#offer"` を**実際のカート/申込URL**（例: colorme等）に置換。別タブなら `target="_blank" rel="noopener"`。

### ② 計測タグ（GTM/GA4）の本体を挿入
CTAクリック時に `dataLayer.push({event:"cta_click", cta:...})` を発火していますが、**GTM/GA本体が未挿入のため現状は計測されません**。
- 対応: GTM（推奨）または GA4 のスニペットを `<head>`（GTMは`<body>`直後にも）追加。
- GTMを入れれば `cta_click` イベントをそのままトリガーに使えます（`cta`値で各ボタンを判別可）。

### ③ 特商法・プライバシー・お問い合わせの確定
フッターのリンクは外部（`colorme-repeat.jp`）で、**未確定項目を含みます**。
- 対応: 正式な特商法表記／プライバシーポリシー／問い合わせ先を確定し、リンク先/内容を差し替え。

### ④ og:image を絶対URLに（任意・優先度低）
`<meta property="og:image">` が相対パス（`assets/01_fv.png`）。SNSシェアのサムネを正しく出すには**公開ドメインの絶対URL**に。

## 実装メモ
- 追従CTA（`.sticky-cta`）: FVが画面内の間は隠れ、スクロールで表示（IntersectionObserver）。価格表示 `¥14,800（税込¥16,280）` はHTMLテキスト。
- FAQ: `<details>` のネイティブアコーディオン（JS不要）。
- 注記（景表法の打消し表示）: 温度表示画像（`08_thermo`）直下・FAQ・フッターの3箇所に `.note` で実装済み。**文言・配置は法務確認の上で調整可**。小さくし過ぎると打消し表示として弱くなる点に注意。
- CSSキャッシュ対策で `style.css?v=...` のクエリを付与。**CSS更新時はこの版数も上げる**こと。
- レスポンシブ: `.lp-shell` は最大幅480pxのスマホ特化。768px以上は中央寄せ表示。
